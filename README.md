# Node GraphQL Server
Playground for GraphQL Server.  Uses the Starwars API (https://swapi.co/)

Built with,
- Babel v7.2
- Node v11.4.0
- Express v4.16.4

## Setup
To install dependancies
```
$ npm i
```

## Server
To start server
```
$ npm start
```

## ESDocs
To create docs from src
```
$ npm run docs:create
```
### Open created docs
### Mac
```
$ npm run docs:open-osx
```
### Linux
```
$ npm run docs:open-nix
```
